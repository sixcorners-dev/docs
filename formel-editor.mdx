### **1. Worum geht's?**

Der Formel-Editor ermöglicht dir, mathematische Ausdrücke direkt im Textfeld sauber zu setzen. 

Er ermöglicht dir:

- optische Hervorhebung im Text (z.B. Materialhinweise)
- mathematische Schreibweise von gängigen Zeichen
- **Sonderzeichen** für z.B. Musik, Chemie, Physik, Mathematik (α, β, → , \<, = …)

  ![image.png](/images/image.png)

### 2. Wie starte ich die Formeleingabe?

Du kannst Formeln auf drei Arten eingeben:

| **1) Quick Versinon**<br />→ nachträgliche optische Hervorhebung im Text                                                                                                                                            | **2) Simple Version**<br />→ Direkteingabe eines Formelbefehl mittels Forat-Option via Maus-Eingabe                                                                                                                                                                                                                                                          | 3) **Pro Version** <br />→ Direkteingabe der Formel mittels Tastenbefehl \$...\$                    |
| :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| **Wofür geeignet?**<br />-**optische Hervorhebung**  von Textabschnitten<br />- Umwandeln einfacher Befehle in Formeln                                                                                              | **Wofür geeignet?**<br />- **einfache Formeln** (a + b = c) mittels gängigen Tastatursymbolen<br />- **komplexe Formeln** mittels code-Befehlen<br />- Eingabe von **Sonderzeichen**                                                                                                                                                                         | **Wofür geeignet?**<br />- wie 1), nur mittels direkter Tastatureingabe                             |
| **Wie geht’s?**<br />→ Schreibe einen normalen Text <br />→ markiere einen Teil davon <br />→ Dreipunkt-Menu <br />→  „Formel“-Button (**√ )**. <br />→ Der Text wird in eine Formel umgewandelt.                   | **Wie geht’s? **<br />→  Öffne eine neue Zeile im Editor<br />→ Tippe ein Slash ein  **/...** <br />→ wähle die Formel-Option aus **√ - **<br />→ Beginne deine Text- oder Formeleingabe direkt im grauen Kasten_(der LateX Schriftzug ist nur Platzhalter) _ <br />→ starte mittels Backslah **\\ ** (Command+Shift+7) die Formel-Befehle (siehe Kapitel 3) | **Wie geht’s?**<br />Formelmodus manuell starten: \$…\$ für inline oder \$\$…\$\$ für Blockformeln. |
